# The Shadowing Aftermath

## Description
The Shadowing Aftermath is a 2D top-down action adventure game 
developed with Unity. Players explore interconnected rooms, battle 
challenging enemies, and defeat progressively stronger bosses to 
unlock new abilities. The game emphasizes fast reactions, pattern 
recognition, and strategic use of unlocked abilities across three 
challenging levels.

## Features
- **Multiple Levels:** 3 progressively challenging levels
- **Combat System:** Fast-paced action combat with enemies
- **Boss Encounters:** Unique boss battles at the end of each level
- **Ability Unlocking:** New special abilities earned after each boss defeat
- **Room Exploration:** Interconnected rooms with combat areas, locked doors, and keys
- **Enemy AI:** Pattern-based enemy behavior with increasing difficulty
- **Main Menu & Victory Screen:** Polished UI for game flow
- **2D Top-Down Perspective:** Classic dungeon-crawler gameplay style

## Game Mechanics
- **Movement:** WASD or Arrow keys to move the player
- **Combat:** Attack enemies using mouse click or combat button
- **Abilities:** Special abilities unlock after defeating bosses
- **Keys & Doors:** Collect keys to unlock doors and progress
- **Boss Battles:** Strategic boss encounters with unique attack patterns
- **Health System:** Player and enemy health management
- **Level Progression:** Complete rooms to progress to next level

## Levels Overview

### Level 1
- Introduction to basic combat mechanics

### Level 2
- More complex room structures

## How to Play
1. Start from Main Menu
2. Progress through Level 1, defeating enemies and the level boss
3. Unlock new ability after boss defeat
4. Use new ability strategically in Level 2 and Level 3
5. Defeat final boss to achieve victory

## Game Controls
- **WASD / Arrow Keys:** Move character
- **Mouse / Combat Button:** Attack enemies
- **Space:** Use special ability (after unlocking)
- **ESC:** Pause game / Return to menu

## Installation & Setup
1. Download or clone this repository
2. Open the project in Unity (2019.4 LTS or later)
3. Ensure all assets are in correct folders
4. Open the Main Menu scene
5. Press Play to start the game
